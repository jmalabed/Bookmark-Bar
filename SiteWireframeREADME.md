## Resource Wireframes
### Resource Show Page
These pages will allow users to view the library of resources that are available for use. This is the core functionality of the site.
![Resource Show Wireframe](https://github.com/jmalabed/project1/blob/submaster/wireframe/pr1-resourcesShow.png)
### Resource New Page
This page will pop up when new resources are added. This helps our site be collaborative and open for users.
![Resource New Wireframe](https://github.com/jmalabed/project1/blob/submaster/wireframe/pr1-newresource.png)
### Edit Resource Page
This edit page is user-login protected and protects the site from being altered by those who do not have an account.
![Resource Edit Wireframe](https://github.com/jmalabed/project1/blob/submaster/wireframe/pr1-edit-resource.png)
## Blog Wireframes
The blog branch of the site provides background for the resources and can give them context. The author of the blog post is also approachable for questions on topics that are relevant in the blog post.
### Blog Home Page
![Blog Home Wireframe](https://github.com/jmalabed/project1/blob/submaster/wireframe/pr1-blog.png)
### View Blog Page

### Edit Blog Page
